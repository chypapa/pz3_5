Логика приложения

Логика приложения - это суть приложения. Что оно делает. За логику отвечает программирование. Это может быть программирование на стороне клиента или программирование на стороне сервера. Каждый из способов имеет свои достоинства и недостатки.

Интерфейс приложения

Интерфейс приложения создается в браузере. Технологии, которые за это отвечают - это HTML и CSS. Лобое серверное приложение в конечном итоге представляет данные браузеру именно с использованием этих технологий.

Обеспечивающие системы

Для работы приложения должен быть развернут Web-сервер. Также часто для организации данных на стороне используестя СУБД. Данные между клиентом и сервером передаются с помощью протокола HTTP.
